# D.COM Intranet Web

D.COM 인트라넷 리뉴얼 프로젝트의 React 프론트엔드 레포지토리입니다.

## Project Overview

기존 PHP, jQuery 기반 D.COM 인트라넷을 React 기반으로 리뉴얼하여,
동아리 운영에 필요한 계정 관리, 족보 아카이브, 정보 공유 게시판, 공지사항,
활동 사진 앨범, 마이페이지, 관리자 기능을 제공하는 것을 목표로 합니다.

## Tech Stack

- React
- JavaScript / TypeScript
- REST API
- CSS / Tailwind CSS

## Related Repository

- Server: `Dcom-KHU/intranet-server`

## Branch Convention

- `main`: 안정 버전
- `feature/*`: 기능 개발
- `fix/*`: 버그 수정
- `docs/*`: 문서 수정
- `chore/*`: 설정 및 기타 작업

예시:

```bash
feature/login-page
feature/signup-page
fix/login-error
docs/readme-update
chore/project-setup
